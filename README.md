# 🧠 Machine Learning with Riya

This repository contains my journey of learning Machine Learning from scratch by combining **theory and practical implementation in a single notebook for each concept**.

---

## 📌 Approach

Each topic is covered in a structured Jupyter Notebook that includes:

* Concept explanation
* Intuition and examples
* Mathematical understanding
* Code implementation
* Visualizations and insights

---

## 📂 Structure

Each folder represents a machine learning concept:

* `concept_name.ipynb` → Theory + Code + Experiments

---

## 🚀 Topics Covered

* Linear Regression
* Logistic Regression
* Decision Trees
* Random Forest
* KNN
* and more...

---

## 🎯 Goal

To build a strong understanding of Machine Learning by learning, implementing, and documenting concepts in a clear and practical way.
